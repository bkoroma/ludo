# Ludo MVP v4.1

This update adds separate cosmetics for the board and the Ludo pieces.

## Added in v4.1
- Table skin shop
- Ludo token skin shop
- Independent selection for table and token cosmetics
- Board rendering changes based on selected table theme
- Token rendering changes based on selected Ludo skin
- Profile migration from old single-skin format

## Cosmetic categories
### Table skins
- Classic Table
- Wood Table
- Emerald Table
- Royal Table

### Ludo skins
- Classic Tokens
- Marble Tokens
- Neon Tokens
- Gold Tokens

## Run
```bash
npm install
npm start
```

Then open:
`http://localhost:3000`
